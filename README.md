## Project Overview
This repository contains the deliverables for the CSE0001 Digital Literacy course. It demonstrates foundational competencies in digital presence, cybersecurity hygiene, version control, and professional online communication. 

## Local Environment Setup & Execution
This project consists of documentation, markdown configuration files, and visual assets. To evaluate and review this project via the command line, follow the execution steps below:

### 1. Clone the Repository
    git clone https://github.com/shivansh-prashant/digital-literacy-project.git
    cd digital-literacy-project

### 2. Verify Directory Structure
    ls -la

### 3. Execute/Read Markdown Files (Example: Cybercrime Case Study)
    cat task-5-cybercrime/casestudy.md

*(You can replace `task-5-cybercrime/casestudy.md` with any other `.md` or `.txt` file path in the repository to read its contents directly in the terminal).*

### 4. View Project Report
To view the comprehensive final report, open the `report/Project_Report.pdf` file using any standard system PDF viewer.
